# Airplane Tracker
## Описание:
Консольное приложение для отслеживания самолетов в реальном времени. 
Программа позволяет получить данные о воздушном движении в выбранной стране,
отфильтровать их по различным критериям (страна регистрации, высота) и 
вывести топ-N самолетов по высоте полета. Все полученные данные сохраняются 
в локальный `JSON-файл` для последующего использования.
## Функционал:
* Приложение использует `API OpenSky Network` для получения информации о самолетах, 
находящихся в воздушном пространстве выбранной страны.
* С помощью `Nominatim API (OpenStreetMap)` определяется географический прямоугольник `(bounding box)`
для поиска самолетов в границах страны.
* Возможность отфильтровать полученные данные по стране регистрации самолета и/или диапазону высот полета.
* Вывод списка топ-N самолетов, отсортированных по высоте полета в порядке убывания.
* Вся информация о самолетах автоматически сохраняется в файл `data/airplane_info.json` для ведения истории.
## Установка:
Клонируйте приложение из GitHub: 

```
git clone https://github.com/kiravolkova7-code/airplane_tracker.git
```

## Зависимости:
Для работы приложения требуется библиотека requests для выполнения HTTP-запросов к API.
В консоли выполните следующий код:
```
poetry add requests 
```
## Тестирование:
Качество кода проверяется с помощью модульных тестов, написанных с использованием фреймворка `pytest`.
Тесты находятся в директории `tests/`.
Чтобы запустить тесты, выполните команду в корневой директории проекта:
```
pytest
```
Тестовый набор покрывает следующие сценарии:

1. Обработка ошибок пользователя:
* Проверка сценария, когда пользователь вводит название несуществующей страны. 
Тесты гарантируют, что программа корректно обрабатывает эту ситуацию и 
выводит понятное сообщение об ошибке.
* Проверка валидации введенных координат. Тесты подтверждают, что программа не позволяет 
выполнить запрос с некорректными данными.

2. Взаимодействие с `API` (мокирование запросов):
* Тесты проверяют, что классы-клиенты `(NominatimApiClient, OpenSkyApiClient)` формируют 
правильные `URL-адреса` и параметры запросов к внешним сервисам.
* Проверяется корректная передача параметров (например, координат и времени) в `API OpenSky`.

3. Логика классов и валидация данных:
* Тестируется создание объекта `AirplaneInfo` как с полными, так и с неполными данными. 
Проверяется, что объект инициализируется корректными значениями по умолчанию.
* Проверяется работа методов преобразования: `to_dict()` и `from_dict()`.
* Тестируются магические методы сравнения `(__eq__, __gt__)`, чтобы убедиться, 
что самолеты сравниваются корректно на основе высоты полета.

4. Работа с хранилищем данных `(JSON)`:
* Проверяется базовый функционал: добавление информации о самолете в хранилище и 
последующее извлечение по различным критериям.
* Тестируется поиск с использованием нескольких критериев одновременно.
* Проверяется защита от дубликатов: система не позволяет добавить один и тот же самолет дважды.

## Использование:
1. Запустите главный скрипт из корневой директории проекта:
```
python main.py
```
2. Следуйте инструкциям в консоли:
- Введите название страны, по которой вы хотите получить данные.
- Уточните область поиска, введя координаты (можно использовать значения по умолчанию).
- Укажите количество самолетов (N) для вывода в топе по высоте.
3. Просмотрите результаты в консоли. 
Информация о самолетах будет также сохранена в файл `data/airplane_info.json`.
